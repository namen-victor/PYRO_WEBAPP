# Contributing

## Branching
- feature/<short-name>
- fix/<short-name>
- docs/<short-name>

## Commit messages
- Use imperative mood: "Add X", "Fix Y"
- Reference issues when relevant

## PR checklist
- [ ] Typecheck passes (`npm run typecheck`)
- [ ] Lint passes (`npm run lint`)
- [ ] Build passes (`npm run build`)
- [ ] Tests pass (`npm test`)

## Local setup
- Copy `env.example` to `.env.local`
- Install deps: `npm install`
- Run dev: `npm run dev`
